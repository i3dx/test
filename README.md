# PONG NEO - 火龙觉醒版

一个现代化的霓虹风格乒乓球游戏，带有酷炫的视觉效果和特效系统。

![Game Preview](preview.png)

## 功能特点

### 核心玩法
- 🏓 经典的乒乓球对战玩法
- 🤖 三种难度等级的 AI 对手（休闲/标准/大师）
- ⚡ 火龙特殊招式系统（每局30次高速球）
- 🎯 首先获得 10 分者获胜

### 视觉效果
- 🌈 霓虹赛博朋克风格界面
- 💥 粒子爆炸特效系统
- ✨ 结束动画特效（胜利/失败不同主题）
- 🔥 火龙球模式发光拖尾效果
- 📳 碰撞屏幕震动反馈

### 交互方式
- ⌨️ 键盘：W/S 或 ↑/↓ 控制移动
- ⌨️ 空格键：释放火龙高速球（游戏进行中）
- 📱 触摸屏：滑动控制
- 🖱️ 鼠标：点击按钮开始/重置

## 技术栈

- **HTML5 Canvas** - 游戏渲染
- **Vanilla JavaScript** - 游戏逻辑（无外部依赖）
- **CSS3** - UI 样式和动画
- **Web Audio API** - 音效生成

## 快速开始

### 方式一：直接打开
双击 `index.html` 文件即可在浏览器中运行。

### 方式二：使用本地服务器

```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server -p 8000
```

然后访问 `http://localhost:8000`

## 游戏控制

| 按键 | 功能 |
|------|------|
| `W` / `↑` | 球拍上移 |
| `S` / `↓` | 球拍下移 |
| `空格键` | 释放火龙高速球（仅有球时可用） |
| `触摸滑动` | 球拍移动（移动端） |

## 游戏规则

1. 使用球拍将球击回对方场地
2. 对方未能接球时得 1 分
3. 首先获得 **10 分** 的玩家获胜
4. 每局限用 **30 次** 火龙高速球
